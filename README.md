# Kinetic Vanguard - Vidish Srivastava

A high-velocity, tactical interactive portfolio built with **React**, **TypeScript**, and **Tailwind CSS v4**. Feature-rich "Kinetic Archive" design for technical architects and AI leads.

## ⚡ Quick Start

### Prerequisites
- Node.js 20+
- npm

### Development
1. **Clone & Install**
   ```bash
   git clone https://github.com/ulolol/ulolol.github.io.git
   cd ulolol.github.io
   npm install
   ```

2. **Launch Neural Link (Dev Server)**
   ```bash
   npm run dev
   ```
   Access at `http://localhost:5173`

## 🛠️ Tactical Architecture

- **Core**: React 18, Vite 6, TypeScript
- **Styling**: Tailwind CSS v4 (High-performance engine)
- **Icons**: Lucide React (Tactical grade)
- **Motion**: Framer Motion (Fluid transitions)

## 📂 Project Structure

```text
ulolol.github.io/
├── src/
│   ├── components/      # UI Modules (Sidebar, Header, Terminal)
│   ├── data/            # Static datastores (linkedin.json)
│   ├── pages/           # Tactical views (Maps, Talents, Quests)
│   ├── App.tsx          # Main neural interface
│   └── main.tsx         # Entry point
├── public/              # Static assets (favicons, profile)
├── index.html           # Source template
└── package.json         # Deployment protocols
```

## 🚀 Deployment

The site is hosted on GitHub Pages. Note that the `deploy` script moves built files to the root.

```bash
npm run deploy
```
