# UI Overhaul Project

## Vision
Transform Vidish Srivastava's portfolio from a generic 2021-era design to a distinctive, modern, quantum-inspired aesthetic that reflects the owner's expertise in AI/ML, quantum computing, and systems programming.

## Current State
- React 18 + Vite + Tailwind CSS portfolio
- Generic blue-purple gradient design
- System fonts only
- Dated card hover effects and pill-shaped tags
- Missing accessibility features

## Target State
- Dark OLED-optimized theme with neon accents
- Custom typography (Archivo + Space Grotesk)
- Glassmorphism and quantum glow effects
- Bento-style layouts
- Full accessibility compliance

## Success Criteria
- [ ] Unique visual identity distinct from generic portfolios
- [ ] WCAG AA accessibility compliance
- [ ] Smooth 60fps animations
- [ ] Responsive across all breakpoints
- [ ] Chrome Lighthouse score >90
